How it works?

It is a process which involve these steps:

  • - Website takes in any random picture.
  • - Then, it will process, by discovering relevant words that the image contains with the word set for which the model is trained for. It will then make a meaningful sentence out of it.
  • - It will display the photo as well as generates the relevant caption below it.

Recurrent Neural Network (RNN) are increasingly being used as encoding-decoding frameworks for machine translation. My objective is to replace the encoder part of the RNN with a Convolutional Neural Network (CNN). Thus, transforming the images into relevant input data to feed into the decoder of the RNN. The image will be converted into a multi-feature dataset, characterizing its distinctive features. The analysis will be carried out on the popular Flickr8K dataset and coco dataset.
